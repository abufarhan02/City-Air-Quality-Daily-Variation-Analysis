# City-Air-Quality-Daily-Variation-Analysis
This City Air Quality Daily Variation Analysis is the Python Based Air Quality Index Analysis the given AQI Data with Charts.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Descriptive statistical analysis
- Line chart visualization
- Peak pollution hour detection

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the script:
   python src/aqi_analysis.py

## Results
The project identifies peak pollution hours and visualizes AQI variations throughout the day.
